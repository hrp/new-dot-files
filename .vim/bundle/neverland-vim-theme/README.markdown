Neverland Vim Theme
====================
All Vim themes suck.

256 colors, CLI and GUI.

The GUI conversations is done by Sam Lidder. 

![screenshot](http://perl.japh.se/devel/neverland-vim-theme/neverland.png)

Installation
============

Copy the themes to $HOME/.vim/colors
